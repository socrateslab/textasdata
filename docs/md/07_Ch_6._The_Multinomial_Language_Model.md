# 第六章 多项分布语言模型

多项分布语言模型
本书所介绍的模型建立于两类基础之上：概率基础与算法基础。本章我们将考察多项分布（multinomial distribution），它建立在上一章所介绍的词袋（bag of words）表示之上，并为本书将要考察的众多概率模型提供了基础。在下一章中，我们将借助向量空间模型（vector space model）给出另一种算法框架。
在概率模型中——统计学领域中最常见的模型——我们通过概率来讲述数据是如何生成的这样一个故事。这一数据生成过程建立在未知参数或未观测到的潜变量（latent variables）之上，我们可以利用实际观测到的数据来推断它们。例如，在分析《联邦党人文集》（Federalist Papers）的例子时，我们会讲述这样一个故事：每位作者以不同的速率在给定文本中生成功能词（function words）。在提出这一故事之后，我们可以利用已有归属的《联邦党人文集》来反推这一过程，并推断出每位作者未知的速率。这反过来又使我们可以通过将推断出的速率与真实的、未归属的文本进行比较，来计算汉密尔顿（Hamilton）或麦迪逊（Madison）撰写某篇有争议的论文的概率。
这些模型之所以有吸引力，是因为它们清晰地陈述了模型在何种情况下能够理想地发挥作用（即当关于数据生成过程的假设是准确的时），并且通过修改数据生成过程，它们很容易扩展到新的情境。或许最为重要的是，对概率模型的设定使我们可以调用统计学中大量现成的步骤来进行优化与不确定性量化。Blei（2014）对这一风格的模型给出了易于理解的概述，他主张一个"构建模型—计算参数—批判结果—然后重复"的循环。
这些概率基础有别于算法方法，后者规定了一系列步骤，通常以一个待优化的目标函数的形式出现。最好将其视为描述相似事物的另一种语言。例如，许多模型，如线性回归，既可以从概率视角（正态线性模型）来描述，也可以从算法视角（最小化误差平方和）来描述。这两种基础之间的差异往往被夸大了。一个概率模型隐含着一个目标函数，而许多目标函数也隐含着一个算法模型。
我们首先引入多项分布，这是一种常用于计数数据的流行分布。然后我们描述它在基础概率语言模型——预测词序列的模型——中的应用。接着我们展示如何通过构建层次模型来扩展这一简单设定。这种构建层次模型的策略将构成本书后面出现的概率聚类、混合（admixture）与分类模型的基础。

### 6.1 多项分布

6.1  多项分布
我们首先构建一个与语言生成并不相符的模型，但它与我们在上一章所发展的文本的词袋表示相一致。我们从一个高度简化的设定开始：我们只有一个由三个词构成的词汇表（cat、dog、fish），并且每个文档只包含一个词元（token，即一个词型（type）的实例）。我们将用长度与词汇表相同的向量来编码每个词元，该向量只包含一个 1（指示所属类别），其余元素均为 0。因此我们用以下向量来表示每个词，它们也恰好是三种可能的唯一文档：
cat = ( 1 , 0 , 0 ) dog = ( 0 , 1 , 0 ) fish = ( 0 , 0 , 1 )
在机器学习中这被称为"独热编码"（one-hot encoding），不过社会科学的研究者可能更熟悉"虚拟变量"（dummy variable）这一术语。
当每个文档只有一个词元长时，我们可以将该文档视为从一个类别分布（categorical distribution）中抽取。类别分布为各个类别的实现赋予一个概率，使得文档 W i , ∼ Categorical( μ)，其中 μ 是一个包含每个词型各自概率的向量。例如，假设 cat 文档出现的可能性是 dog 和 fish 的两倍；我们便有向量

### 6.3 正则化与平滑

6.3  正则化与平滑
回想一下，模型令人困惑地将约翰·杰伊（John Jay）撰写有争议文本的概率赋为零。这是因为归属于约翰·杰伊的五篇文章中，他从未使用过词 man；因此，μ J , Man 的极大似然估计为 0。当词 man 出现在我们的有争议论文中时，我们得出结论：该有争议论文属于杰伊是一个零概率事件。这似乎是一个过于强烈的结论。在现实中，我们只是拥有的杰伊文档较少，因此更可能是出于偶然他从未用过那个词，而不是他将永远不使用那个词。
语言的性质意味着罕见词相当常见，因此这一特定问题也很常见。一种有价值的方法是在我们的估计量中加入正则化（regularization），它将估计量拉向某个特定的取值。正则化是一种非数据的信息或约束，被加入模型以将估计量拉向某个特定的取值。其思想是，这可以帮助防止过拟合，例如避免得出杰伊写 man 这个词的概率为零的结论。
解决这一问题最常见的方法之一是，简单地向每位作者每个词型的计数加上某个固定的数量 α。这实质上添加了伪数据（pseudo-data），并将估计量推离零。在我们的连续例子中，我们可以设想设置 α = 1，这将把杰伊的词计数变为 (83, 1, 2)。这将 man 的概率从 0 变为 .02，并将杰伊生成有争议文本中那些词的概率从 0 变为 .008——虽然不太可能，但不再是不可能。α 对杰伊估计的影响大于对汉密尔顿或麦迪逊的影响，因为当真实数据量增大时，伪数据量保持不变，因而它对估计量的影响越来越小。这种正则化策略被称为拉普拉斯平滑（Laplace smoothing）。
拉普拉斯平滑是存在于概率模型之外的附加修补，但我们也可以通过先验分布将正则化纳入模型本身。这实质上赋予参数（如 μ）一个属于它自己的数据生成过程。其结果是贝叶斯层次模型（Bayesian hierarchical model），它以与拉普拉斯平滑大致相同的方式引入正则化。不过请回想，参数 μ 必须落在单纯形上，因此要加入先验，我们需要一个分布，使得每次抽取都落在单纯形上的一个点。这类分布中最常见的是狄利克雷分布（Dirichlet distribution），我们将在下一节讨论。

### 6.5 结论

6.5  结论
在本章中，我们为基于多项分布（Multinomial distribution）的词袋模型奠定了概率基础。我们还展示了如何使用多项分布的共轭先验——狄利克雷分布——来构建一个简单的层次模型。这些模型之所以有吸引力，是因为一旦被设定，它们便使研究者能够借助统计学与计算机科学中大量现成的结果，这些结果使得计算成为可能并建立理论性质（Bishop, 2006; Murphy, 2021; Gelman et al., 2013)[^1][^2][^3]
