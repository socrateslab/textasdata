# 第28章 结论

结论

结论
本书是关在社会科学中使用"文本作为数据"工具的指南。文本作为数据在社会科学中的兴起，与社会科学家所处的一个新的丰裕与机遇时代相吻合。凭借分析大量文本的能力，社会科学家得以在以往不可能的规模上获得洞见。与此同时，大量新的统计模型与计算工具使得分析这些庞大的新数据集比以往任何时候都更加容易。
在本结论章中，我们首先回顾本书的核心论点，并就如何扩展这些原则提供指引。在社会科学中，数据方法在聚焦于社会科学任务时才能发挥最佳作用。因此，尽管我们在分析文本作为数据时使用的许多方法起源于计算机科学，我们认为这些方法在应用于社会科学时需要被重新定位。而且关键的是，当我们评估这些方法时，应用于社会科学问题的方式将有所不同。我们还认为，文本作为数据的推断最好以迭代和顺序的方式进行。尽管社会科学家常常以演绎方式推进，我们认为，当我们从数据集中归纳地构建证据基础时，基于文本的推断会得到改善。并且，当我们顺序地开展工作时，我们能够改进我们所定义的概念、我们所进行的测量，并最终改进我们从数据中得出的推断。最后，我们主张，在使用机器学习模型时，我们应当抱有一种基线的怀疑态度——对我们不知道真实的数据生成过程持不可知论（agnosticism）。我们所使用的模型都在提供复杂文本数据的独特概要方面具有价值。但由于文本数据的表征方式与实际语言运作方式之间的复杂关系，我们避免依赖假设我们拥有正确模型的验证。相反，我们认为，聚焦于我们核心任务的验证对于评估模型在社会科学任务中表现是否良好至关重要。
虽然我们聚焦于文本，但本书的核心原则远不止适用于文本作为数据。事实上，我们将文本作为数据的方法视为更普遍地适用于许多其他新数据源——包括图像、声音和视频。与文本一样，这些数据源在社会科学的定性研究中也有着悠久的使用历史。但要在大规模上处理这些数据一直很困难。计算机科学中开发的新计算工具使得在大规模上处理这些数据源容易得多。不过，同样的问题出现了。计算机科学家在处理图像、视频和声音时所聚焦的任务，并非社会科学家感兴趣的任务。我们的框架提供了如何将这些新工具应用于社会科学推断的指南。
文本作为数据的方法，以及更广义的机器学习，被誉为对社会科学家的一大进步。在许多方面，我们同意这一点。新的数据源和方法使研究者能够提出并回答关于我们先前无法在规模上研究的社会互动的问题。这些方法并非万灵药。我们甚至怀疑是否存在解决社会科学问题的银弹（silver bullet）。无论新方法多么令人印象深刻，社会科学家仍将面临因果推断、预测以及仔细定义概念以进行测量这些艰巨挑战。我们以一段讨论作结：为何在开发新方法时保持谦逊，对于避免过度承诺所带来的失望至关重要。

28.1  如何在社会科学中使用文本作为数据
28.1.1  聚焦社会科学任务
文本作为数据在社会科学中的快速扩张，建立在计算机科学、统计学和语言学领域一个庞大且极为成功的文献基础之上。这些方法极其有用，但其目标是要解决与社会科学不同的问题。例如，无监督文本作为数据方法是为信息检索任务而开发的，其主要目标是从海量信息中挑出一根针——这一任务通常以留出数据（held-out data）的预测准确性来评估。但社会科学家想要刻画整个草堆，而非找到那根针，并且他们很少将预测作为无监督文本的指标。事实上，结果往往表明，在预测上表现更好的模型，在提供社会科学洞见方面有时反而更差。出于这个原因，我们围绕应用这些方法论的社会科学理由来组织方法。
社会科学的任务遵循研究项目的轨迹。在研究项目开始时，我们从事发现（discovery），确定文本的新概念化以获得对底层模式的洞见。基于这些组织方式，我们从事测量（measurement），或将观测分配到各个类别。最后我们从事推断：要么是因果推断，即我们询问某一特定干预会产生什么效应；要么是预测，即我们在样本外基于数据作出估计。
这种基于任务的方法不仅适用于文本作为数据的工作。事实上，我们认为，将这些工作明确地围绕此类任务来组织，有助于形成富有成效的研究议程。我们发现，在指导研究生首次如何开展严肃研究时，将研究解构为这些阶段可以使令人生畏的研究项目变得易于管理。它也有助于分解现有项目，以理解是什么在驱动推断。
我们也想明确指出，尽管本书没有专门设章讨论理论，我们将社会科学理论视为研究过程不可或缺的组成部分。理论可以提示概念化，可以激发测量，并有助于我们对因果效应的解释。虽然有人曾提出大数据时代将意味着理论的终结，我们认为是恰恰相反。理论对于解释统计模型的结果至关重要，而随着数据增多，我们有机会汇总我们的推断，并拥有更多程式化事实来构建模型。

我们认为，当我们的推断是迭代和顺序的时，我们从数据中获益最多。我们这样认为有基本的理由。从数据迭代地构建使我们能够更好地理解我们的模型在特定情境下如何应用，并针对我们的特定应用调整特征表征和其他模型特征。顺序地开展工作使研究者能够在不同研究之间累积证据。当我们在进行分析时发展出假设，顺序地思考意味着我们能用新数据和下一个研究设计来检验那些假设。
除了对分析者的好处之外，我们认为顺序方法对于研究的实际开展方式更为诚实。尽管学者们常常将论文写得仿佛观点是演绎产生的，但我们的经验是很少如此。相反，观点是在论文写作过程中迭代地、顺序地发展起来的。更清楚地说明研究过程，将使研究如何建立在先前努力之上、以及未来工作如何能建立在当前论文之上变得更加明确。
我们也认为，对研究的迭代和顺序方法有助于解决"fishing"问题。实验中一个主要的担忧是，研究者会在分析中作出决策，以制造存在统计显著结果的印象，而真实的差异为零。一种应对方法是预注册（pre-registration）。当然，这可能是有用的。但它几乎无助于解决研究者发表具有显著结果论文的激励。作为替代，我们认为对研究的顺序方法可以创造激励，使研究者去重复他们的发现（并让其他研究者重复他们的发现）。

当我们处理文本数据时，我们的观点是，我们应当怀疑自己是否曾拥有正确的模型。语言极其复杂，甚至我们对语言的表征都需要大量的简化假设。因此，我们在全书所使用的模型显然是错误的，但其中一些是有用的。这意味着这些模型对于测量感兴趣的量、提示新的概念化、甚至促进因果推断都是有用的。
但这也意味着，当我们进行模型选择时，我们不应仅关注模型拟合统计量。这是因为模型拟合很少针对社会科学家使用模型的特定方式进行优化。相反，我们认为，为了选择最有用的模型，我们要从事与模型将如何使用紧密相关的验证。而这常常涉及将"人置于回路之中"（humans in the loop）来评估模型。将人保留在回路中之所以重要，不仅是为了确保我们的测量有效、模型有意义，而且正如我们在全书中所强调的，是为了运用对情境和实质知识的理解，确保我们以务实且符合伦理的方式应用模型、开展研究，并始终牢记那些写下文本的人也是人，而不仅仅是数据点。

### 28.2 将我们的原则应用于文本数据之外

28.2  将我们的原则应用于文本数据之外
在本书中，我们聚焦于文本作为数据，但我们的论点更普遍地适用于其他数据源。事实上，我们认为，随着新的数据源变得可用，以及新方法从其他领域引入，关键是我们应当追问这些方法将如何在政治科学中被使用。思考社会科学家如何最佳地使用这些新数据源，我们认为，将引导其在社会科学中最富成效的应用。
在相关的新数据上已经取得了令人难以置信的进展。例如，越来越多的学者使用机器学习技术从图像数据中作出社会科学推断。例如，Torres（2019）考察了抗议者在新闻媒体照片中是如何被刻画的。更一般地，我们认为学者们早已认识到图像对社会互动的力量。研究图像的机器学习方法提供了在规模上研究群体如何被刻画、并理解这些刻画如何影响公众的机会。
类似地，在规模上研究实际政治言论方面已经取得了令人难以置信的进展。当然，我们认为说话的方式很重要。民主党人广泛称赞巴拉克·奥巴马以一种为其修辞增添生机的节奏和语调发表振奋人心的演讲的能力。同样，罗纳德·里根以其轻松、真诚且带着愉悦的神情传达最重要信息的方式激励了保守派。社会科学家在分析言论在社会互动中的作用方面取得了重要进展。例如，Kaufman、Kraft 和 Sen（2019）已经表明，最高法院大法官的说话方式可以预测他们在案件中的裁判倾向；Knox 和 Lucas（2021）
