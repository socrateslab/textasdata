# Chapter 17. An Overview of Supervised Classification

An Overview of Supervised
Classification
本章为利用有监督分类（supervised classification）实现测量自动化提供了一个默认的操作流程。有监督分类使用一组被标注到各个类别中的文档，来建立一个统计模型（有时称为"分类器"，classifier），该模型将文档中的词语与标签联系起来，并将该模型应用于其他未标注文档，以获得可比较的标签。我们对有监督分类的整个过程进行了概述——从标注训练集、训练模型，到验证结果。在随后的三章中，我们将更深入地探讨这三个组成部分中的每一部分。
在有监督分类中，目标是建立文档集中的特征（feature）与研究者定义的更一般类别和概念之间的映射。到目前为止，我们已经考虑了定义映射函数的两种重要情形：一种是分析者选择一个特定词语并将其计数作为感兴趣测量的代理变量；另一种是映射函数由研究者通过词典显式定义。而在有监督分类中，我们通过训练一个算法，利用一组已编码文档的样本，从文本中预测感兴趣的类别，从而从数据中学习映射函数。我们定义一个模型，使得对文档 i 的预测 y i ̂ 由函数 f μ ( W i , ) 生成，其中 W i , 是文档的表示（representation），μ 是模型的参数。然后，我们通过观察输入 W 和输出 y 的随机样本（称为训练样本，training samples）来学习模型的参数。这本质上将测量问题转化为一个预测任务，利用统计模型来预测分析者本人在标注时会赋予的标签。这种方法相当通用，并且具有一个优点：一旦学习了模型的参数，对额外文档进行分类几乎不耗费成本。
在这种方法中，人类负责提供三类信息：训练文档的标签、输入的表示，以及要学习的模型类别。拥有标签通常意味着分析者已经在脑海中形成了一种编码方案和一个映射函数。我们目前假设分析者随机选取训练文档的样本，并以完全准确的程度将标签 y 赋予所选文档。
第二步是输入的特征表示。这是整个过程中一个重要却常被低估的环节。所选的表示应与分析者所从事的任务类型相匹配。贯穿本节的一个反复出现的主题是：我们用来预测标签 y 的特征，远比模型类别重要得多。这一点在文献中常被忽视，文献作者往往试图提供适用于分析者可能遇到的任何映射函数的通用建议。然而，我们每次都会毫不犹豫地选择更好的特征而非更好的模型。出于阐释的目的，我们使用词项计数（term counts）的向量 W i , 作为我们的表示，但我们同样可以轻松选择基于词嵌入（word embeddings）、短语，或从文本中提取的广泛其他特征的表示。
第三步也是最后一步，是选择要学习的模型类别。可供选择的模型种类繁多，令人眼花缭乱；但所有这些不同的选项都提供了不同的信息共享方式。如果我们对特征如何映射到输出不做任何假设，那么我们只能对与训练集中观察到的样本完全相似的文档进行分类。不同的模型类别定义了学习到的函数如何从输入表示映射到标签。我们可以选择相对简单的模型，这类模型通常易于解释，且训练所需数据相对较少，但可能不够灵活，无法预测人类的标签。更复杂的模型对数据更饥渴，对分析者而言可能相对不透明，但往往能处理复杂的预测问题。

### 17.1 Example: Discursive Governance

17.1  Example: Discursive Governance
为了对有监督分类进行概述，我们借用 Gillion (2016) 对美国政治官员关于种族的话语分析中的一个例子。尽管政治战略家认为，政治官员应当对其围绕政策的言论去种族化（deracialize），以吸引最广泛的选民群体，但 Gillion (2016) 认为，种族在修辞中的作用具有超出选举和政策制定过程之外的影响。Gillion (2016) 认为，讨论种族对他所称的"话语治理"（discursive governance）具有重要含义——在这一过程中，修辞不仅直接影响选举和政策，也影响公众和政策制定者如何设定议程、设计、讨论、实施和评估政策。他主张，不讨论种族会降低种族不平等在选民眼中的显著性，并使关于政策对此种不平等影响的重要信息被排除在政策制定过程之外。
作为其证据基础的一部分，Gillion (2016) 测量了总统和国会议员所发表的大量声明中关于种族的讨论——这些声明包括"总统辩论、竞选筹款演讲、告别演讲、就职演说、对全国的演讲、每周电台讲话、新闻发布会、国情咨文、对国会的其他演讲、地方演讲（例如毕业典礼演讲和市政厅演讲）、签署声明，以及对外国立法机构和联合国大会的讲话"（第 33 页）。他还包括了众议院的所有议院演讲、国会记录（Congressional Record）中的所有文档，以及《总统公开文件》（Public Papers of the Presidents）的所有卷册。为了将这些文本划分为声明，Gillion (2016) 以段落作为分析单位，最终得到来自总统的 912,466 个段落（1955–2012）和来自国会议员的 230 万个段落（第 104–112 届国会）。
接下来，Gillion (2016) 着手测量文档中的"种族相关话语"（race-related discourse），他将其定义为"政治家表达的、主要与代表性不足的种族或族裔少数群体相关，或明确使用带有种族含义的术语的政治对话或修辞"（第 33 页）。显然，Gillion (2016) 收集的文档数量过于庞大，无法由人工编码者手动编码。但正如他所指出的，词典也同样不够充分。符合这一种族相关定义的段落可能实际上并未使用"race"一词，即使它们讨论的是种族议题。其他词语，如"busing"（校车接送），可能仅在特定的语境中才与种族相关。因此，Gillion (2016) 决定采用人工编码与有监督机器学习相结合的方法，将每个段落编码为种族相关或非种族相关。在此，我们沿着 Gillion (2016) 测量的发展，通过三个步骤来展开：1）创建训练集；2）利用有监督学习对文档进行分类；3）检查性能。

### 17.2 Create a Training Set

17.2  Create a Training Set
有监督分类的第一步是创建训练集，我们将在第 18 章中详细讨论这一点。人工标注者将文档的一个子集标注到研究者定义的类别中。在 Gillion (2016) 中，两名标注者阅读了较大样本中的 12,300 个段落，并根据 Gillion (2016) 提出的具体定义，指出它们是否与种族相关。创建训练集时必须考虑两个主要因素。首先，有监督分类器能否将训练集中的模式泛化到测试集，取决于训练集中的文档是如何从更大集合中抽取的。一个不具代表性的训练集可能会导致一个在泛化到新文档时产生偏差的分类器。使用文档的随机样本作为训练集，是确保训练文档代表样本其余部分的最简单方法。Gillion (2016) 为了确保代表性做得更进一步，他创建了文档的分层随机抽样，以确保它们能代表时间周期。在某些情况下，随机抽样是不可能的；我们将在第 18 章更详细地讨论这些问题。
第二个要考虑的因素是人类编码者自身的准确性。即便是最简单的分类方案，不同的编码者也可能会有不同理解。清晰地传达类别定义非常重要，这不仅是为了今后的工作，也是为了在训练集被更广泛地应用之前确保其准确性。正如我们在第 18 章详细讨论的那样，我们建议至少由两名人工标注者对文档进行编码，随后评估编码者间信度（intercoder reliability），即两名编码者将文档归入同一类别的比例。Gillion (2016) 报告其两名标注者之间的编码者间一致率为 97%（第 164 页）。当标注者意见不一致时，由他本人做出最终判定。

### 17.3 Classify Documents with Supervised Learning

17.3  Classify Documents with Supervised Learning
有监督分类的下一步是选择一个分类器，以学习训练集中标注的特征与类别之间的映射。这首先涉及选择特征的表示方式，我们在第 2 部分：表示（Representation）中讨论过这一点。我们建议部分地借助领域专业知识来选择表示，仔细考虑那些最可能对各类别提供信息的特征类型。许多分类器还包含特征选择（feature selection）组件，它会移除分类器认为不相关的特征。Gillion (2016) 使用词袋（bag of words）方法来表示文档，去除了标点符号、数字、空白字符和停用词（stop words）（第 164 页）。下一步是选择用于学习映射的模型。可供选择的模型有很多，其中一些我们将在第 19 章详述。此外，模型可以组合成分类器集成（ensembles of classifiers），研究者使用许多不同的模型对每个文档的最终标签进行"投票"。这正是 Gillion (2016) 用来将文档编码为种族相关或非种族相关的方法。

### 17.4 Check Performance

17.4  Check Performance
有监督分类的最后一步是检查分类器的性能。在此，研究者应当留出一部分训练数据——将其排除在第 (2) 步所述的模型拟合之外——用于验证。在拟合模型之后，研究者将该模型应用于这一留出数据集，并比较模型在人工标签上的表现。在这一验证步骤中有一些需要考虑的因素，我们将在第 20 章详述。首先，为了防止对某一特定验证数据集过拟合（overfitting），研究者通常使用交叉验证（cross-validation），将数据分割、拟合模型，并多次评估准确性，然后对性能指标取平均。Gillion (2016) 采用的就是这种方法，报告的总统文档集准确率为 0.77，国会文档集准确率为 0.66。1
另一个需要考虑的因素是性能指标本身。虽然准确率（accuracy）——即正确分类的总数除以验证集中的总数——是一个很自然的性能度量，但高准确率可能掩盖糟糕的表现，尤其是当数据中的类别相当偏斜时。我们在第 20 章详述了其他性能指标，包括精确率（precision）和召回率（recall），它们可以为分析者提供更多关于性能的信息。

### 17.5 Using the Measure

17.5  Using the Measure
一旦 Gillion 检查了性能，他便可以着手将分类器应用于测试集。通过将分类器应用于其数据集中的所有声明，Gillion (2016) 可以利用这一新测量来理解：与种族相关的总统声明频率在不同政府任期内如何变化，以及国会议员关于种族的声明如何与地理分布相关联。图 17.1。来自 Gillion (2016) 的图 2.1 和图 2.3。左图显示了随时间的推移与种族相关的总统声明所占比例。右图显示了在众议院发言中发表关于种族声明的众议员地理分布；颜色越深的地区表明声明更为频繁。
在图 17.1 中，我们根据有监督测量重现了 Gillion (2016) 中的两幅图。左图显示了随时间的推移与种族相关的总统声明所占比例。令人惊讶的是，与其他总统相比，奥巴马总统的种族相关声明相当少，Gillion (2016) 认为，奥巴马可能因其作为黑人总统的身份而感到压力，从而尽量减少对种族的讨论。右图显示了国会议员声明中种族讨论的地理分布。Gillion (2016) 指出，这些讨论大多集中在美国东南部。
正如我们将在后续章节中讨论的，一旦我们将分类器应用于测试集，就必须仔细思考如何在后续分析中使用它。我们利用分类器创建的测量包含测量误差，这些误差可能来自人工编码的错误，也可能来自分类器的错误。理想情况下，我们将找到方法将这种测量误差传递到最终分析中，并考察该误差是否以可能影响我们分析的方式系统地存在。

### 17.6 Conclusion

17.6  Conclusion
在本章中，我们对有监督分类的过程进行了概述。借助 Gillion (2016) 中种族相关话语的测量，我们从宏观视角展示了研究者如何利用由人工编码者创建的训练集来构建自定义测量——首先编码训练集，然后利用有监督分类器对文档进行分类，最后评估该测量的性能。在接下来的三章中，我们将详细深入探讨这三个步骤中的每一个。
