# 第27章 文本作为混杂因素

文本作为混杂因素
文本不仅可以在实验中作为结果变量（outcome）或干预（intervention）使用，还可以用于在观测性分析中控制变量。在此我们借鉴 Roberts、Stewart 和 Nielsen（2020）的研究，描述文本如何成为因果分析中的混杂因素，以及我们如何使用文本分析来控制这类混杂。1
文本作为混杂变量在社会科学应用中频繁出现。举一例，Maliniak、Powers 和 Walter（2013）提供的证据表明，在国际关系领域，由女性撰写的文章被引用的可能性低于由男性撰写的文章，原因在于作者的（感知）性别。作者即使在控制了学术资历（如终身教职和学校的排名）以及文章层面的协变量（如发表期刊、主题、视角和文章年龄）之后，仍然发现了这种关系。一篇女性作者的文章是否比男性作者的文章被引用的可能性更低？在这种情况下，作者的性别是处理（干预），结果变量是引用的数量。然而，由于性别并非随机分配，我们不能简单地将男性和女性之间引用次数的均值差异作为性别对引用因果效应的估计。最重要的是，女性在国际关系中所写的主题与男性不同。文章文本内容可能与处理（干预）（感知性别）和结果变量（引用）都相关；因此，文章的文本混淆了性别与引用之间的关系。结果，Maliniak、Powers 和 Walter（2013）花费了论文大量篇幅，使用手工编码的、描述文章内容的类别作为控制变量来解决这个问题。
文本作为混杂因素在分析社会数据时出奇地常见。立法法案的内容可能混淆否决威胁与国会的重新定位之间的关系。学生的大学录取材料或推荐信可能既与其种族或性别相关，又与其被大学录取的概率相关。国际协定的内容可能混淆贸易与国际合作关系之间的关系。
控制数值型变量相对直接——研究者可以使用基本的线性回归或匹配（matching）来调整混杂因素。然而，控制文本的内容则要困难得多。研究者不能将文章中每一个独特的词都纳入匹配或回归，因为词的数量很可能远多于观测数。此外，研究者并不想完全控制全部文本——理想情况下，他们只想针对那些混淆处理（干预）与结果变量的词或词簇进行调整。
使用像文本这样的高维协变量进行调整，带来了"维度灾难"（curse of dimensionality）的挑战。对文本进行调整的困难在于，弄清楚文本的哪些方面应当被纳入，哪些方面不应当被调整。很多时候，研究者事先并不知道文本的哪些方面同时与处理（干预）和结果变量相关。他们可能不知道男性和女性在国际关系中对主题的讨论有何不同，以及这些差异的哪些方面可能与引用计数相关。因此，在这些情况下，研究者必须估计正在混淆该关系的文本方面。
在数学上，混杂问题的结构如下。如第24章，我们从一个包含 N 个单元的数据集开始。每个单元 i 被分配处理（干预） t i，处理组取值为 1，控制组取值为 0。在潜在结果框架下，结果变量 y i 在单元 i 被处理时取 y i ( 1 ) 的值，在单元 i 为控制时取 y i ( 0 ) 的值。由于我们考虑的是观测性数据的情况，t i 不是随机分配的，处理组和控制组可能不可比。一种常见做法是在 N × P 矩阵 X 中对我们的 N 个观测匹配 P 个处理前协变量，以改善处理组和控制组内协变量分布的相似性，这一条件称为平衡（balance）。当 X 包含了所有混杂来源时，我们可以使用匹配来实现平衡（X i ╨ t i），从而能够对处理组上的平均处理效应进行无偏估计。
在大多数匹配应用中，X 是低维的，P ≪ N。采取可观测变量上的选择（selection on observables）策略，我们假设条件可忽略性（conditional ignorability）：t i ╨ y i ( 0 ) , y i ( 1 ) | X i。在此假设下，在处理组间平衡已观测协变量 X 的分布为我们提供了一种估计感兴趣因果效应的方法。在某些情形下，使可观测变量选择成立的 X 的 P 个变量对研究者是已知的，因为处理（干预）分配机制是透明的。
然而，在我们所考虑的、不仅要对 X 进行调整、还要对文档-词项矩阵 W 进行调整的情形下，这两个矩阵合并后的维度 P + J 非常大。在这种情况下，J（语料（库）的词汇量）可能远大于 N（观测数）。此外，我们未必知道 W 的哪些方面与处理（干预）和结果变量相关。我们认为文本中的某些词会影响处理（干预），但不确定是哪些。如果我们尝试在所有词上匹配，除非两篇文本具有完全相同的词频，否则我们将找不到任何匹配——而随着 W 维度的增大，这种可能性变得微乎其微。
我们的方法是估计 W 中变量的一个低维概要，我们可以用它来处理混杂，这是一个针对混杂因素的 g 函数，而非像之前那样针对处理（干预）和结果变量。因此，文本匹配要求如下：

在本节中，我们探讨几种不同的寻找合适 g 以利用文本数据调整混杂的方法——既有基于回归的调整，也有基于匹配的调整。虽然我们只关注少数几种简单的方法，但我们注意到这是一个快速发展的领域。自 Roberts、Stewart 和 Nielsen（2020）于 2015 年最初发布其关于文本匹配的论文以来，调整文本混杂的新进展迅速跟进。Mozer 等人（2020）改变了用于匹配文本的表征和距离度量，以评估数百种不同的文本匹配方法。而 Veitch、Wang 和 Blei（2019）创建了一种利用词嵌入（word embeddings）来调整混杂的方法。综述见 Keith、Jensen 和 O'Connor（2020）。
在全章中，我们使用来自 Maliniak、Powers 和 Walter（2013）的例子，研究国际关系（IR）学科中的性别化引用偏差。如果女性撰写与男性完全相同的文章，却被引用得更少，那么针对 IR 中女性学者引用偏差的证据将最为有力。然而，男性和女性往往在国际关系中写不同的主题、使用不同的方法，并持有不同的认识论立场。由于这些因素可能影响引用计数，女性较低的引用计数可能反映的是对某些主题和方法的偏差，而非针对女性本身。3 Maliniak、Powers 和 Walter（2013）[^1] 利用 TRIP 期刊文章数据库中的信息，对每篇文章的广泛子领域、所涵盖的议题领域、一般方法、范式4 和认识论进行控制，以应对这一挑战。在本应用中，我们试图使用自动化方法而非手工编码来解决这一混杂问题。5

### 27.1 针对文本混杂因素的回归调整

27.1  针对文本混杂因素的回归调整
在社会科学数据分析中，调整混杂最常见的方式是将混杂变量纳入回归。因此，对像文本这样的高维变量进行调整，最简单的扩展就是将其纳入回归，但使用收缩（shrinkage）方法来为手头的应用选择最佳的回归变量子集。如第19章所讨论的，惩罚回归估计量（如岭回归（Ridge regression）和 Lasso）可以选择对结果变量具有高度预测力的协变量，并将较不重要的协变量收缩或剔除。如果这些估计量选择了与混杂相关的词进行调整，那么我们就可以用它们来调整文本混杂因素。
我们采取这第一种非常简单的方法，对文章文本进行调整以估计性别的影响。我们估计一个 Lasso，将每篇文章的引用次数对文章的词项-文档矩阵以及 Maliniak、Powers 和 Walter（2013）在其分析中纳入的所有其他协变量（如文章年龄、终身教职和期刊）进行回归。我们还纳入一个指示处理（干预）的变量：文章是否仅由女性作者撰写。我们不对处理（干预）系数施加惩罚，因为那是感兴趣的系数列。我们使用交叉验证来选择其余系数的最优惩罚。随后我们将 Lasso 模型选择的词纳入预测引用的线性回归中。
我们希望 Lasso 模型能够选择重要的、对结果变量具有高度预测力的词和协变量，从而使我们能够舍弃许多不感兴趣的词。图 27.1 显示了最终纳入模型的词，因而也是对引用高度预测的词，包括与资助相关的词（如 Minerva）、与方法论相关的词（如 cause-effect 和 zscore），以及与实质内容相关的词（如 surgenc 和 democ）。特定作者的引用似乎对预测引用也很重要，例如 weingast 和 shepsl。该模型还提示 Maliniak、Powers 和 Walter（2013）编码的某些特定协变量很重要，例如文章是否发表在《美国政治科学评论》（政治科学的主要阵地）上，或者文章是否具有实证主义（positivist）框架，这两者都预测更高的引用数。
对女性作者身份的估计系数为负，表明在调整了 Lasso 选择的词以及其他混杂因素之后，女性作者身份与引用之间呈负相关关系。
使用 Lasso 是在文本回归中选择一组词的相对直接的方法，但它有一些重要的缺点，这些缺点说明了使用高维混杂进行因果推断的复杂性。第一个缺点是，我们调整混杂的能力受到所选文本表征（词计数）和线性模型组合的限制。例如，这排除了词与词之间的交互作用，但有可能当 minerva 与 democracy 这类词一起使用时会产生大量引用，而与 autocracy 一起使用时引用很少。第二个缺点源于使用 Lasso 正则化回归来解决潜在混杂变量的高维问题。正如 Belloni、Chernozhukov 和 Hansen（2014）所解释的，虽然 Lasso 选择了作为良好预测变量的变量，但它可能并未选择最佳的混杂因素。7 这是我们一般性观点的又一例证：当引入为其他目的设计的工具时，需要一些细心来使其适应感兴趣的任务。Belloni、Chernozhukov 和 Hansen（2014）提出了一种替代策略，涉及两个不同的 Lasso 回归——一个预测结果变量，一个预测处理（干预）——然后使用所选变量的并集进行最终的回归调整。
一种替代方法是从文本的较低维表征入手，这样正则化就不那么必要。例如，我们可以使用主题模型将每篇文档表示为若干主题的组合。随后我们可以估计性别对引用的影 响，同时控制主题内容。如果主题足以捕捉混杂，这一策略将高效得多，因为主题的维度通常远小于词汇量。这一策略可能带来的效率提升，类似于在假设成立时朴素贝叶斯（Naive Bayes，第19章）的好处。

对于控制数据的低维表征，可以采取许多方法，分析者会希望选择他们认为在混淆处理（干预）与结果变量关系中最重要的方法。例如，可以使用主成分分析（PCA）或奇异值分解（SVD）来替代主题，以控制解释文档方差的词簇。然而，请注意 PCA、SVD，甚至基本的主题建模方法，在创建文本的降维时都没有利用结果变量或处理（干预）的信息。例如，图 27.2 中主题的最可能相关词并不包含 minerva 这个词，尽管这个词对引用具有高度预测力。在选择协变量时纳入关于结果变量或处理（干预）的信息，可能有助于确保模型捕捉到混杂因素。

### 27.2 针对文本的匹配调整

27.2  针对文本的匹配调整
使用回归调整的一种替代方法，是利用文本分析在处理组和控制组单元之间寻找写有相似主题或使用相似词的匹配。Roberts、Stewart 和 Nielsen（2020）认为，匹配的优势在于可比的单元更容易被考察——匹配允许用户找到匹配方法认为相似的文档，阅读它们，并评估这些观测是否彼此构成良好的反事实。文本模型可能会失败，因此，正如我们在全书中所强调的，这种方法使分析者能够判断被匹配的单元是否足够相似，以至于处理（干预）分配可以说是"近似随机"的。这种人工验证补充了匹配中常用的其他正式平衡检验。

可以用于调整文本混杂因素的匹配方法数以百计，各有优劣。Roberts、Stewart 和 Nielsen（2020）开发了一种方法——主题逆回归匹配（Topical Inverse Regression Matching，TIRM），它整合了他们所看重的匹配策略的两个特征：建模处理（干预）分配与粗化（coarsening）。为此，他们使用结构主题模型（Structural Topic Model，STM）联合估计主题——将词粗化为主题组——以及文档层面的得分——以指示每篇文档被处理（干预）的可能性。通过将处理（干预）作为 STM 模型中的主题内容协变量，他们可以联合估计主题和文档层面的得分。随后他们在估计的主题和文档层面得分上进行匹配。
为了建模处理（干预）分配，TIRM 通过估计模型的一个投影来创建类似于倾向得分匹配（propensity score matching）的方法，该投影是关于处理（干预）概率的词计数信息的充分降维。遵循 Taddy（2013），Roberts、Stewart 和 Nielsen（2020）[^2] 推导了词计数中关于处理（干预）信息的充分降维。通过在主题模型的背景下估计这一文档层面得分，该投影代表了未被主题所承载的、关于处理（干预）的信息。
为了在粗化的主题组上进行匹配，Roberts、Stewart 和 Nielsen（2020）从 STM 模型中提取文档的主题内容。由于处理（干预）作为内容协变量被纳入时会出现主题-处理（干预）交互作用，同一主题在处理组和控制组下可能具有不同的估计词分布，这带来了一个小麻烦。作者通过向 STM 添加最后一步估计，将所有控制组文档的主题比例重新估计为仿佛它们被处理（干预）了一样，从而确保主题无论处理组/控制组差异如何都具有可比性。这一选择与一个（局部）处理组平均处理效应的估计量一致。随后他们在 STM 投影和 STM 估计的主题比例上都进行匹配，这确保了匹配既在主题上相似，又在主题内具有相似的处理（干预）概率。
Roberts、Stewart 和 Nielsen（2020）将 TIRM 应用于 Maliniak、Powers 和 Walter（2013），在算法的 STM 部分指定 15 个主题以恢复宽泛的主题。作为比较，他们还仅使用投影、以及仅使用主题创建了匹配样本。随后他们使用 TIRM 匹配样本重新估计了 Maliniak、Powers 和 Walter（2013）的模型，发现性别在引用上的差异比最初报告的更为显著。他们发现，与尽可能相似的、由男性或混合性别群体撰写的文章相比，女性撰写的文章平均少被引用 6.5 次。图 27.1 包含了他们提供的、状态不同但文章相似的匹配对示例。

### 27.3 结论

27.3  结论
文本测量了如此多的社会现象，但在用于进行因果推断的分析中目前很少被使用。在本书的这一部分，我们建立了一个框架，用于思考如何在因果推断中使用文本，并阐明了研究者在将文本作为处理（干预）、结果变量或控制变量使用时可能遇到的潜在陷阱。我们在全章中包含了许多在因果推断中使用文本的例子，以展示利用文本研究社会科学的因果关系的巨大潜力。
当然，因果识别是困难的，在因果分析中使用文本并不能解决因果推断的任何核心问题。相反，将文本用于因果推断可能增加而非减少复杂性。我们敦促读者，如果他们此前未学习过因果推断，应利用该领域大量的文献（Morgan and Winship, 2015; Imbens and Rubin, 2015; Hernán and Robins, 2020）[^3][^4][^5]。此外，我们建议分析者利用训练集/测试集拆分来在每个实验内部模拟重复验证，并尽可能在新的总体和时段中重复他们的实验。
