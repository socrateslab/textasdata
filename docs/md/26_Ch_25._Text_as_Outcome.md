# Chapter 25. Text as Outcome

Text as Outcome
文本作为结果变量的设定在分析上是直接的。具体而言，g 提供了一个从文本——现在记为 y，以强调它是所研究的结果变量——到所关注的低维结果变量 π 的映射。假设某个特定的 g，我们希望估计平均处理效应（ATE）。使用我们在第24章讨论的训练/测试划分方法，g 在训练文档上发展，我们用下标 I 来标示它。而 ATE 则在测试集 O 上估计，我们用下标 O 来标示。假设 SUTVA、可忽略性和正向性（来自第24章的假设1–3），并采用 Egami 等人 (2018) 的符号，我们可以将 ATE 定义为：
A T E ̂ = ∑ i ∈ O I ( t i = 1 ) g I ( y i ( 1 ) ) ∑ j ∈ O I ( t j = 1 ) − ∑ i ∈ O I ( t i = 0 ) g I ( y i ( 0 ) ) ∑ j ∈ O I ( t j = 0 ) .
换句话说，通过在训练文档 I 上学习 g，我们将其视为 O 中数据的一个固定变换，并像平常一样进行估计。Egami 等人 (2018) 提供了一个识别证明。
在本章中，我们回顾两个使用文本作为结果变量的例子。首先，我们讨论 Egami 等人 (2018) 进行的一项关于对移民态度的调查实验。其次，我们重点介绍 Franco, Grimmer, and Lim (2018) 的工作，他们估计了总统演讲对媒体报道的影响。在每种情况下，作者都使用训练/测试划分来发现 g，然后依次估计因果效应。此外，Egami 等人 (2018) 使用序贯实验来迭代地精炼 g，然后才最终估计 ATE。

### 25.1 An Experiment on Immigration

25.1  An Experiment on Immigration
Egami 等人 (2018) 研究对个体犯罪历史的 awareness（认识）如何影响对移民的态度，使用了一项调查实验。基于 Cohen, Rust, and Steen (2004) 的一项较早研究，他们随机分配调查受访者阅读描述一名非法移民的两种不同假设情境。在两种情况下，该人都非法进入美国。然而，在一种情况下，情境描述该人有犯罪史。
处理组情境如下：
"一名28岁的单身男性，另一国公民，被判非法进入美国。在此次犯罪之前，他曾服过两次刑期均超过一年的监禁。其中一次前科是暴力犯罪，且他已被遣返回其祖国。"
对照组情境如下：
"一名28岁的单身男性，另一国公民，被判非法进入美国。在此次犯罪之前，他从未被监禁过。"
受访者随后被问了一个开放式问题，要求他们描述美国政府在这种情况下应该怎么做。
"这名违法者应该被送进监狱吗？"（回答选项：是、否、不知道），但紧接着被问到："为什么或为什么不？请至少用两句话描述美国政府应该对此人采取什么行动（如果有的话）以及为什么？"
分析的目标是估计情境对受访者关于美国政府应采取何种行动的写作的影响。然而，为了估计这一点，作者必须发展一个 g 函数，将开放式回答的文本映射为一个可用的所关注量。

Egami 等人 (2018) 结合使用了训练/测试划分和序贯实验来发现 g 并估计因果效应。首先，他们使用第13章描述的结构主题模型（Structural Topic Model）分析了 Cohen, Rust, and Steen (2004) 提供的数据 (Roberts et al., 2014)[^1]。然后，他们在2017年7月的 Mechanical Turk 实验中，做了若干设计变更后复制了该实验。最后，他们在2017年9月再次复制了该实验。在每种情况下，他们都使用训练/测试划分来发展 g，并仅在测试集上估计 ATE。
我们首先展示他们运行的最后一个实验的结果。表25.1显示了11个主题中每个主题最可能的词、Egami 等人 (2018) 给出的主题标签，以及估计的效应大小和置信区间。如你所见，主题从倡导遣返（主题2和3）到要求获取更多案件信息（主题5）不等。显然，处理对受访者所写的主题有影响。被告知该移民有犯罪史，会增加受访者关于遣返的写作量，并降低受访者主张该人应被允许留在美国的概率。

### 25.2 The Effect of Presidential Public Appeals

25.2  The Effect of Presidential Public Appeals
在我们的第二个将文本用作因变量的例子中，Franco, Grimmer, and Lim (2018) 考察了总统在黄金时段电视上公开呼吁对后续媒体报道的影响。大量文献就总统影响公众舆论及其在新闻中如何被报道的能力给出了混合的结果。混合结果的一个原因是，先前的研究使用了不同的研究设计和不断变化的证据类型来评估总统呼吁的效应。为了分离出呼吁的效应，Franco, Grimmer, and Lim (2018) 将注意力限制在黄金时段电视报道期间总统发表的公开声明上。然后，他们收集了演讲前一周和后一周内十大报纸对总统的报道，以及同一时期的广播电视新闻转录文本。其结果是收集了近160,000篇提及总统的故事，横跨总统的公开呼吁。
Franco, Grimmer, and Lim (2018) 使用这些数据来探究总统对他们从媒体获得的报道类型的影响。为了进行这一评估，Franco, Grimmer, and Lim (2018) 使用结构主题模型来估计一组主题，使用10%的新闻报道和转录文本作为训练集，其余90%作为测试集。这一划分确保训练集中有足够信息来发现显著主题，测试集中有足够内容来推断是否存在处理效应。
基于这一主题拟合，Franco, Grimmer, and Lim (2018) 随后识别每次总统公开呼吁的显著主题。利用这一被识别的显著主题，他们随后估计总统的演讲在后续新闻媒体中对该显著主题的效应。
图25.1中的左图显示了总统演讲在两种不同情境下对主题报道的效应：1) 当总统向国会发出呼吁时（黑色、顶部线条）；2) 当总统发表声明传达某个重大世界事件发生时（灰色、底部线条）。利用这些百分比，Franco, Grimmer, and Lim (2018) 通过比较演讲开始前一天和结束后一天报道所占的份额，来推断演讲的效应。
左图显示，当总统发出公开呼吁时，他们大幅增加了报道。在呼吁国会之后，总统获得了比演讲前即刻比率高出3.4个百分点的该主题报道。底部图显示了为什么在总统宣布重大世界事件之后评估总统演讲的效应会有问题。在总统宣布一个事件之后，提及总统的文章中该主题的报道增加了七个百分点。当然，其中一部分增加应归因于所发生的事件，而非总统的演讲，因此该估计将总统的演讲与该事件在世界上额外获得的报道混为一谈了。
也许并不令人意外，我们看到宣布事件的报道增加比呼吁持续得更久。
